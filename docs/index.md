# Python & Data Analytics for CA Professionals

**Welcome!** I'm Aayush, your Facilitator for this training programme at ICAN.

This course is designed specifically for **Chartered Accountants and finance professionals** who want to harness the power of Python to automate, analyse, and visualise financial data — with zero prior coding experience required.

---

## What You Will Learn

| Module | Topic | Key Skills |
|--------|-------|------------|
| 01 | **Python Basics** | Variables, data types, loops, functions, tax calculator |
| 02 | **NumPy** | Arrays, vectorized math, NPV/IRR, portfolio analysis |
| 03 | **Pandas** | DataFrames, pivot tables, VLOOKUP replacement, Excel I/O |
| 04 | **Matplotlib** | Line, bar, pie, waterfall charts, dashboards |
| 05 | **Seaborn** | Statistical charts, heatmaps, outlier detection |

---

## How to Use This Course

!!! tip "Getting Started"
    Work through the modules **in order** — each one builds on the previous.
    Every module includes:

    - Clear explanations with finance-focused examples
    - Runnable code cells you can experiment with
    - Mini exercises after each section
    - Full solutions at the end

!!! note "Tools Required"
    - Python 3.8 or above
    - Jupyter Notebook or JupyterLab
    - Libraries: `numpy`, `pandas`, `matplotlib`, `seaborn`

    Install everything at once:
    ```bash
    pip install numpy pandas matplotlib seaborn jupyterlab
    ```
